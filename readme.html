Inventory Site (Netlify‑ready)

This folder contains a static inventory browser built from your Excel file. To keep adding data from different offices, you have two easy storage options on Netlify:

  1. Netlify DB (Postgres via Neon) – Best for structured, multi-user data. Create tables like items and locations, then add a small Function netlify/functions/inventory.mjs that exposes GET/POST for the front-end.
  2. Netlify Blobs – Simple key/value JSON. Good for quick uploads of the latest inventory.json per site or per office.

If you want, I can add the Function and DB schema in a follow-up so you can push updates directly from the website (no spreadsheets needed).
